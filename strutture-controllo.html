Strutture di Controllo PHP

Istruzione IF - Condizioni

L'istruzione if permette di eseguire codice solo se una condizione è vera. È possibile aggiungere else per specificare cosa fare quando la condizione è falsa.

<?php
    $eta = 18;
    
    if ($eta >= 18) {
        echo "Sei maggiorenne!";
    } else {
        echo "Sei minorenne!";
    }
?>

Operatori di confronto comuni:

  • ==: uguale a (confronto valore)
  • ===: identico a (confronto valore e tipo)
  • != o <>: diverso da
  • <, >, <=, >=: minore, maggiore, minore o uguale, maggiore o uguale

IF con ELSEIF - Condizioni Multiple

Quando hai bisogno di verificare più condizioni, puoi usare elseif:

<?php
    $voto = 85;
    
    if ($voto >= 90) {
        echo "Ottimo!";
    } elseif ($voto >= 70) {
        echo "Buono!";
    } elseif ($voto >= 60) {
        echo "Sufficiente!";
    } else {
        echo "Insufficiente!";
    }
?>

Ciclo FOR - Iterazione con Contatore

Il ciclo for esegue un blocco di codice un numero specifico di volte:

<?php
    // Stampa i numeri da 1 a 5
    for ($i = 1; $i <= 5; $i++) {
        echo "Numero: " . $i . "<br>";
    }
?>

Componenti del ciclo for:

  • $i = 1: inizializzazione (eseguita una volta all'inizio)
  • $i <= 5: condizione (verificata prima di ogni iterazione)
  • $i++: incremento (eseguito dopo ogni iterazione)

Ciclo FOREACH - Iterazione su Array

Il ciclo foreach è perfetto per scorrere gli elementi di un array:

<?php
    $frutti = array("Mela", "Banana", "Arancia");
    
    foreach ($frutti as $frutto) {
        echo "Mi piace: " . $frutto . "<br>";
    }
?>

Ciclo WHILE - Iterazione con Condizione

Il ciclo while continua finché la condizione è vera:

<?php
    $numero = 1;
    
    while ($numero <= 5) {
        echo "Numero: " . $numero . "<br>";
        $numero++;
    }
?>
