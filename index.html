Introduzione al PHP

Cos'è il PHP?

PHP (Hypertext Preprocessor) è un linguaggio di scripting lato server open source, ampiamente utilizzato per lo sviluppo web. PHP viene eseguito sul server e genera HTML dinamico che viene poi inviato al browser del client.

PHP è perfetto per creare siti web dinamici, gestire database, processare form e molto altro!

Struttura di base di un file PHP

Un file PHP ha estensione .php e può contenere sia codice HTML che codice PHP. Il codice PHP è racchiuso tra i tag <?php e ?>:

<!DOCTYPE html>
<html>
<head>
    <title>La mia prima pagina PHP</title>
</head>
<body>
    <?php
        echo "Ciao Mondo!";
    ?>
</body>
</html>

Elementi fondamentali:

  • <?php ?>: delimitatori che contengono il codice PHP
  • echo: stampa testo o variabili sulla pagina
  • ;: punto e virgola che termina ogni istruzione PHP
  • I file PHP devono essere eseguiti su un server (come Apache, Nginx) con PHP installato
