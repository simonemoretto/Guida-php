Array Associativi in PHP

Cosa sono gli Array?

Un array è una variabile che può contenere più valori. In PHP esistono tre tipi di array:

  • Array indicizzati: usano indici numerici (0, 1, 2...)
  • Array associativi: usano chiavi nominate
  • Array multidimensionali: array che contengono altri array

Array Indicizzati

Gli array indicizzati usano numeri come indici (partendo da 0):

<?php
    // Creazione di un array indicizzato
    $frutti = array("Mela", "Banana", "Arancia");
    
    // Sintassi alternativa (PHP 5.4+)
    $colori = ["Rosso", "Verde", "Blu"];
    
    // Accesso agli elementi
    echo $frutti[0];  // Output: Mela
    echo $colori[2];  // Output: Blu
    
    // Lunghezza dell'array
    echo count($frutti);  // Output: 3
?>

Array Associativi

Gli array associativi usano chiavi nominate invece di indici numerici. Sono perfetti per rappresentare dati strutturati:

<?php
    // Creazione di un array associativo
    $persona = array(
        "nome" => "Mario",
        "cognome" => "Rossi",
        "eta" => 30,
        "citta" => "Roma"
    );
    
    // Sintassi alternativa
    $auto = [
        "marca" => "Fiat",
        "modello" => "Panda",
        "anno" => 2023,
        "colore" => "Bianco"
    ];
    
    // Accesso ai valori tramite chiave
    echo $persona["nome"];     // Output: Mario
    echo $auto["marca"];       // Output: Fiat
?>

Vantaggi degli array associativi:

  • Più leggibili: le chiavi descrivono i dati
  • Facili da manutenere
  • Ideali per rappresentare oggetti o entità
  • Perfetti per configurazioni e impostazioni

Iterare su Array Associativi

Puoi scorrere gli array associativi usando il ciclo foreach:

<?php
    $studente = [
        "nome" => "Laura",
        "corso" => "Informatica",
        "anno" => 2,
        "voto_medio" => 28
    ];
    
    // Iterare solo sui valori
    foreach ($studente as $valore) {
        echo $valore . "<br>";
    }
    
    // Iterare su chiavi e valori
    foreach ($studente as $chiave => $valore) {
        echo $chiave . ": " . $valore . "<br>";
    }
    
    /* Output:
       nome: Laura
       corso: Informatica
       anno: 2
       voto_medio: 28
    */
?>

Modificare Array Associativi

Puoi aggiungere, modificare o rimuovere elementi dagli array:

<?php
    $prodotto = [
        "nome" => "Laptop",
        "prezzo" => 899
    ];
    
    // Aggiungere un nuovo elemento
    $prodotto["marca"] = "HP";
    
    // Modificare un elemento esistente
    $prodotto["prezzo"] = 799;
    
    // Rimuovere un elemento
    unset($prodotto["marca"]);
    
    // Verificare se una chiave esiste
    if (array_key_exists("nome", $prodotto)) {
        echo "Il prodotto ha un nome";
    }
    
    // Metodo alternativo con isset()
    if (isset($prodotto["prezzo"])) {
        echo "Prezzo: " . $prodotto["prezzo"];
    }
?>

Array Multidimensionali

Gli array possono contenere altri array, creando strutture complesse:

<?php
    // Array di studenti
    $classe = [
        [
            "nome" => "Mario",
            "voti" => [28, 30, 27]
        ],
        [
            "nome" => "Laura",
            "voti" => [29, 28, 30]
        ],
        [
            "nome" => "Giuseppe",
            "voti" => [25, 26, 28]
        ]
    ];
    
    // Accedere agli elementi
    echo $classe[0]["nome"];      // Output: Mario
    echo $classe[1]["voti"][0];   // Output: 29
    
    // Iterare su array multidimensionale
    foreach ($classe as $studente) {
        echo $studente["nome"] . ": ";
        
        foreach ($studente["voti"] as $voto) {
            echo $voto . " ";
        }
        echo "<br>";
    }
?>

Funzioni Utili per Array

PHP offre molte funzioni per lavorare con gli array:

<?php
    $numeri = [5, 2, 8, 1, 9];
    
    // Ordinare array
    sort($numeri);           // Ordina in modo crescente
    rsort($numeri);          // Ordina in modo decrescente
    
    // Per array associativi
    $eta = ["Mario" => 30, "Laura" => 25, "Giuseppe" => 35];
    asort($eta);             // Ordina per valore
    ksort($eta);             // Ordina per chiave
    
    // Altre funzioni utili
    count($numeri);          // Conta elementi
    in_array(5, $numeri);    // Verifica se esiste un valore
    array_push($numeri, 10); // Aggiunge elemento alla fine
    array_pop($numeri);      // Rimuove ultimo elemento
    array_merge($arr1, $arr2); // Unisce due array
    
    // Ottenere chiavi e valori
    $chiavi = array_keys($eta);      // Ottiene tutte le chiavi
    $valori = array_values($eta);    // Ottiene tutti i valori
?>

Esempio Pratico: Lista della Spesa

<?php
    $spesa = [
        "Pane" => ["quantita" => 2, "prezzo" => 1.50],
        "Latte" => ["quantita" => 1, "prezzo" => 1.20],
        "Uova" => ["quantita" => 12, "prezzo" => 3.00],
        "Pasta" => ["quantita" => 3, "prezzo" => 0.90]
    ];
    
    $totale = 0;
    
    echo "<h3>Lista della Spesa</h3>";
    echo "<ul>";
    
    foreach ($spesa as $prodotto => $dettagli) {
        $subtotale = $dettagli["quantita"] * $dettagli["prezzo"];
        $totale += $subtotale;
        
        echo "<li>";
        echo $prodotto . " - ";
        echo "Quantità: " . $dettagli["quantita"] . " - ";
        echo "Prezzo: €" . $dettagli["prezzo"] . " - ";
        echo "Subtotale: €" . number_format($subtotale, 2);
        echo "</li>";
    }
    
    echo "</ul>";
    echo "<strong>Totale: €" . number_format($totale, 2) . "</strong>";
?>
